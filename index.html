Glean is a little ( ~ 5 KB ) jQuery reliant plugin that gives you the power of variables where you wouldnt usually be able to get variables. Namly places like ghost or any markdown centric apps.

Okay, so what is it?

Its an incredibly simple, easy to implement jQuery plugin that gives you the ability of using simple variables in your html and or markdown, Think of it like sprinkling spicy pepper on an already great meal.
